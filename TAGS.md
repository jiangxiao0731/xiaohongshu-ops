# 标签数据库

状态：v1.1 数据驱动版（基于120条竞品帖分析 + XHS 2026平台规则修订）

**平台规则：每篇笔记最多10个标签。** 超过10个不会报错但多余标签无效。

更新规则：
- 每次发布后72小时，记录该帖标签组合的表现
- 每两周根据数据淘汰低效标签，补入新标签
- 标注数据来源（假设/竞品数据/自有数据）
- 参与2个以上官方话题活动/周可获得算法加权
