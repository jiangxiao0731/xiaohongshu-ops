# 公司号帖子模板「蕉蕉椒椒（作品集版）」v4.0

**人设：** 个人老师Shaw，NYU IDM毕业，第一人称，活人感优先
**发布节奏：** 修复期 每周1篇（周三晚7:30北京 / 纽约7:30AM，高峰前30分钟）；建量期起 每周2篇（周三+周日）

---

## 核心原则：活人感写法

| 广告感（避免） | 活人感（采用） |
|-------------|-------------|
| 「今天教你3个技巧」 | 「昨天有个学生把项目给我看，第一眼我就知道问题在哪」 |
| 「技巧1：选题要小而具体」 | 「我每次看到『探索人与科技的关系』这种题目，脑子里都在想：说了等于没说」 |
| 「如有需要请私信咨询」 | 「你现在的选题是什么，评论区说说，我来看看值不值得做」 |
| 深色背景大白字机构排版 | 真实工作截图、草稿照片、iPad笔记 |

有效的「活人语气」信号词：「我发现」「每次看到」「上周有个学生」「其实我当时也」「说实话」

无效的「机构语气」信号词：「本人提供」「欢迎咨询」「专业辅导」「成功案例」「XX年经验」

## 类型一：创意技术教程（30%）

**切入点：** 「我最近在研究XX，踩了个坑想记下来」
**竞品均值：** 2867赞藏

**标题公式：** `[具体效果/工具] | 我踩坑后总结的[X]个关键`
示例：`Stable Diffusion出图一直很糊｜问题根本不在模型`

**正文结构：**
```
昨天/上周[具体触发事件，真实场景]。

[问题/现象，第一人称]

我试了X次才发现：[核心洞察，一句话]

→ 第一步：
→ 第二步：
→ 第三步：

你现在做的是什么方向，评论区说说，我来看看。
```

**封面：** 工作截图 / 前后对比 / 草稿界面（非精心排版的信息图）

**标签（10个，平台上限）：**
```
#数字媒体艺术 #创意编程 #TouchDesigner #交互设计 #生成艺术 #新媒体艺术 #作品集辅导 #艺术留学作品集 #交互设计留学 #数字艺术
```

## 类型四：offer案例拆解（10%，修复期≤1篇/月）

**标题格式：** `[学校]录取案例 | [作品集核心概念一句话]`
示例：`UAL交互录取案例 | 把失眠做成了一个生成装置`

**正文结构（严格不超过200字）：**
```
有个学生来找我时，做的是[方向X]。
我们聊了以后发现问题在[具体问题，不写结果]。

→ project 1：[名字] — [研究问题一句话]
→ project 2：[名字] — [研究问题一句话]
→ project 3：[名字] — [研究问题一句话]

这套作品集能走通，关键是[策略逻辑，聚焦方法论]。

有类似方向的评论区说说，我来看看。
```

**封面：** 作品集页面截图（学生授权），不用offer邮件截图

**绝对禁止：** 时间/GPA/录取结果/「辅导后」

**标签（10个，平台上限）：**
```
#作品集辅导 #艺术留学作品集 #交互设计留学 #留学作品集 #交互设计作品集 #作品集 #艺术生留学 #数字媒体艺术 #UAL #offer
```
