# 个人号帖子模板「蕉蕉椒椒」v4.0

**人设：** 创作者Shaw，纯创作维度
**绝对红线：** 不出现 留学/申请/辅导/offer/录取/院校 任何字样
**品牌连接：** 简介仅写 `蕉蕉椒椒（作品集版）`，帖子里绝不提另一个号（矩阵运营是2026年最高风险违规行为）
**发布节奏：** 有内容发，没有不凑数。休眠后重启需先发2-3条休闲暖号帖（展览/日常），再恢复专业内容

## 类型一：TD/创意编程教程（55%）

**自有数据均值：** ~75赞，Mediapipe手势控制帖131赞为最高
**标题公式：** `[X分钟TD教程｜XX效果]` / `[效果描述] | 完整思路`

**正文结构：**
```
[作品效果一句话描述]

实现思路：
1. [步骤1，带具体参数]
2. [步骤2]
3. [步骤3]

[技术要点，1段，带个人判断]

代码/文件评论区
```

**配图/视频：** 成品展示（视频优先）+ 关键节点截图

**标签（10个，平台上限）：**
```
#Touchdesigner #TouchDesigner教程 #创意编程 #生成艺术 #新媒体艺术 #数字艺术 #交互艺术 #互动装置 #视觉艺术 #generativeart
```

## 类型二：AI工具实测（25%）

**自有数据均值：** 161赞（最高），要跟热点工具
**标题公式：** `[工具名]实测 | [真实输出展示/判断结论]`

**正文结构：**
```
[工具一句话介绍]

我测了[X个场景/X次]：
- 好用：[具体场景+截图]
- 不好用：[具体问题]
- 适合：[哪类人/哪类需求]

[Shaw的直接结论，有立场]
```

**要求：** 必须有真实输出展示（截图/视频），不能只是评价

## 类型三：数媒展览/毕业展（20%）

**自有数据均值：** ~60赞（NYU IDM帖83赞，ITP帖46赞）
**切入点：** 纯欣赏视角，不写「可以用来申请」

**正文结构：**
```
[展览/作品名，一句话背景]

[Shaw的具体观察，有立场，不是流水账]

[技术/概念层面的有趣点，1段]

[可选：这个逻辑在创作上的启发]
```
