## 排期逻辑

**按学生做作品集的实际流程排，不是随机轮换。**

26fall 申请截止 2026年9-12月。现在3月底，学生此刻最需要的内容：
- 4月初：还没开始的人最多，最急的是"选什么方向""怎么开始"
- 4月中：开始做了，卡在"开题""研究怎么写"
- 4月底：做了一阵了，需要"排版""怎么整理成完整作品集"

所以排序是：方向选择 → 开题方法 → 实操排版，递进关系。

**两号配合：** 同一周公司号发干货，个人号发创作内容，认知互补。公司号建立"这个老师懂行"，个人号建立"这个人自己也在做"。

### 投流决策树

```
发帖后观察24h自然流量
  >200眼睛 → 不开薯条，或50元小额放量
  50-200 → 开薯条50元x3天诊断
  <50 → 开薯条150元x3天，二次验证限流
薯条曝光 >500 → 没限流
薯条曝光 <100 → 确认限流，执行PLAN应急方案
```
