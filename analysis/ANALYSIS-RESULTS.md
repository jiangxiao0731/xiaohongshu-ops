### 可复制的格式（标✓的三种）

**格式A：案例+作品描述（最适合公司号）**
```
标题：[学校缩写]录取案例 | [作品集核心概念一句话]
示例：「UAL交互录取案例 | 用身体运动可视化城市噪声」
示例：「NYU IDM录取案例 | 把毕业焦虑做成了一个生成装置」
```

**格式B：多offer展示**
```
标题：[方向]+[学校列表]双/三offer，作品集思路分享
示例：「交互设计方向，UAL+米兰理工双offer，作品集全拆解」
```

**格式C：作品集展示型**
```
标题：拿下[学校]的作品集，[核心特点]
示例：「拿下RCA的作品集，全程零代码」
```

### 不要用的格式
- 「恭喜XXX同学拿到offer」→ 广告感强，转化差
- offer邮件截图作封面 → 数据差，压权重
- 「XX机构学员offer捷报」→ 0信任度

## 结论三：账号名建议

### 数据

搜索结果账号类型分布：
- 纯中文：61个（51%）
- 中英混合：34个（28%）
- 纯英文：25个（21%）

### 关键发现

TOP表现账号的命名规律：
- @QROOM交互设计留学（10376赞藏）→ 英文缩写+中文方向
- @RoSSo艺术留学作品集（4156赞藏）→ 品牌名+中文定位
- @SIA艺术留学规划团（3611赞藏）→ 英文缩写+中文定位
- @ACG国际艺术留学（6753赞藏）→ 英文缩写+中文

### 结论：**建议改名**

「No Turn on Red」纯英文，在搜索「作品集辅导」「交互设计留学」时不包含任何关键词，损失搜索曝光。

**建议格式：英文品牌名+中文定位词**
推荐选项（选一个）：
1. `NTBR 作品集辅导` （No Turn on Red缩写+定位）
2. `Shaw Studio 数媒交互留学` （Shaw老师+方向）
3. `No Turn on Red｜作品集` （现名+关键词）

最低成本方案：在现有英文名后面加 `｜作品集辅导` 即可，不需要改名，只需改简介展示名。
